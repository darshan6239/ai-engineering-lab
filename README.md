## ▶️ How to Run Any Project

1. Clone the repo
   ```bash
   git clone https://github.com/darshan6239/ai-engineering-lab.git
   cd ai-engineering-lab
   ```
2. Move into the project folder you want to try
   ```bash
   cd project-folder-name
   ```
3. Follow that project's own README for install and run steps
